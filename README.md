# 🌙 Responsive Navbar with Dark Mode (Tailwind CSS)

A modern responsive navigation bar built using **HTML, Tailwind CSS, and Vanilla JavaScript**.  
It includes a mobile menu, dark/light mode toggle, and Font Awesome icons.

---

## 🚀 Features

- 📱 Fully responsive navbar (mobile + desktop)
- 🍔 Mobile hamburger menu with open/close functionality
- 🌙 Dark / Light mode toggle (saved in localStorage)
- 🎨 Styled using Tailwind CSS utility classes
- ⚡ Smooth UI interactions with JavaScript
- 🔔 Font Awesome icons integration

---

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS
- JavaScript (Vanilla JS)
- Font Awesome 7

## ⚙️ How It Works

### 📱 Mobile Menu
- Clicking the **bars icon** opens the navigation menu
- Clicking the **close icon** hides it

### 🌙 Dark Mode
- Click the moon icon to toggle dark mode
- Theme preference is saved using `localStorage`
- Automatically detects system theme if no preference is set

---

## 💡 JavaScript Features

- Toggle mobile navigation menu
- Dark mode switching with class toggle on `<html>`
- Persist theme using `localStorage`
- Icon switching between moon ☾ and sun ☀
